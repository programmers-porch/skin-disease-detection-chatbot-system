# 🧠 Skin Disease Detection Chatbot (HTML + FastAPI)

## 📌 Overview
This project is a **simple web-based chatbot** for detecting skin diseases using user-input symptoms.  
It uses **HTML, CSS, and JavaScript** for frontend and **FastAPI** for backend processing.

---

## 🚀 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Vanilla)

### Backend
- FastAPI (Python)
- Uvicorn

---

## 🧠 Features

- 💬 Simple chat interface
- 🧾 Symptom-based disease detection
- ⚡ Fast response system
- 🧠 Rule-based matching logic
- 🖥 Lightweight implementation
